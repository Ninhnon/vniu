## Installation & Setup

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/Ninhnon/vniu.git
   cd vniu
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and go to:
   ```
   http://localhost:3000
   ```
